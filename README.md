## Proyectos destacados

### WinRenamer
Utilidad de sistema que permite renombrar cualquier ventana de Windows en tiempo real mediante un hook global WH_CALLWNDPROC. La DLL se inyecta en todos los procesos del sistema y se comunica con el proceso principal por IPC.

**Stack:** C++17 · WinAPI · CMake · Inno Setup

---

### Best Grabador
Grabador de pantalla y audio para Windows con interfaz WinUI 3. Soporta múltiples fuentes de audio simultáneas y perfiles de calidad configurables.

**Stack:** C# · .NET 8 · WinUI 3 · NAudio
